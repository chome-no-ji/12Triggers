# 12Triggers
**[JP]** LLMの長文暴走・誤解・コンテキスト崩壊を全部防ぐプロンプト集

**[EN]** Prompt set to prevent LLM long-windedness, misunderstanding, and context collapse.

> 🇺🇸 **English Users:** Please use [12Triggers_EN.md](12Triggers_EN.md) code.

**LLMの長文暴走・誤解・コンテキスト崩壊を全部防ぐプロンプト集**

1つのプロンプトを貼るだけで、Gemini・GPT・Claude・Grokが全部言うこと聞きます。
外部ツール・API・ファインチューニング一切不要。

## ⚡ 特徴
* **12+1のフラグ監視**：長文回答を事前に予知して阻止
* **完遂強迫の防御**：「全部書いて」「省略なし」という命令すらも、長くなるなら無視して要約
* **詳細アンロック**：要約された後に「詳しく」「続き」と言うだけで、制限を解除して全文出力
* **「時を戻そう」**：ミスった会話をなかったことにして完全巻き戻し
* **「これ余談だけど」**：文脈（コンテキスト）を守ったまま雑談が可能

---

## 📖 使い方 (How to Use)

特別な設定は不要です。**「新しいチャット」**でも、**「今まさに暴走している会話の途中」**でも、いつでも適用可能です。

### ステップ 1: コードのコピー
このリポジトリにある [12Triggers_FULL_PROMPT.md](12Triggers_FULL_PROMPT.md) の中身をすべてコピーしてください。
（ファイルを開いて、右上の □ が重なったアイコン「Copy raw file」を押すと一発でコピーできます）

### ステップ 2: AIに送信
お使いのAI（ChatGPT, Gemini, Claude, Grokなど）のチャット欄に、先ほどコピーしたプロンプトを**そのまま貼り付けて送信**してください。

> **💡 ヒント:**
> 「話が長すぎて読みづらい」「指示を忘れている」と感じたその瞬間に貼り付ければ、AIの挙動がその場で矯正されます。

### ステップ 3: 起動確認
AIが「システムを起動しました」と応答すればセットアップ完了です。

## 🎮 コマンド一覧

会話中に以下のキーワードを使うことで、AIを自在に制御できます。

| コマンド | 機能 | 解説 |
| :--- | :--- | :--- |
| **(自動発動)** | **長文防御 & 要約** | 話が長くなりそうな時は「短く300文字でまとめるね」と要約するか、「長くなるけどいい？」と許可を求めてきます。「全部書いて」と言っても発動します。 |
| **詳しく / 続き** | **制限解除** | 直前の要約に対してこのコマンドを使うと、**リミッターを一時解除**し、本来の長文や詳細データをすべて出力させることができます。 |
| **時を戻そう** | **完全巻き戻し** | 直前の会話を「なかったこと」にします。AIが「〇〇の話をする前に戻しました」と注釈付きでリセットします。 |
| **これ余談だけど** | **文脈保護** | 今の話の流れを凍結して雑談に応じます。「余談終了」で元の話に復帰します。 |
| **システム終了** | **停止** | 12Triggersを停止し、通常のAIに戻します。「オフ」「やめ」でも通じます。 |

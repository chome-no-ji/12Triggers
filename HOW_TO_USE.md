# 12Triggersの使い方（コピペ1発で即起動）

1. 好きなLLM（Gemini / GPT / Claude / Grokなど）に  
   「12Triggers_FULL_PROMPT.md」の内容を**全部コピペ**して送信

2. 最後に  
   「起動して」  
   とだけ送る

→ これだけで即座に「12Triggers」が発動  
   ・長文を完全に防ぐ  
   ・「時を戻そう」で完全リセット  
   ・「これ余談だけど」で選択肢を守る

たったこれだけ。  
外部ツール・API・ファインチューニング一切不要。
